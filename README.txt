DSA Assignment 3 – Group 11
Description

This project includes implementations of a stack, a binary search tree, and a postfix calculator. The goal of the project is to demonstrate understanding of these data structures and their basic operations.

Components
- Array-based stack implementation
- Stack interface and supporting classes
- Binary Search Tree (BST)
- Postfix expression calculator
- Main class for execution and testing

File Overview
- ArrayStack.java – implementation of a stack using an array
- StackInterface.java – defines stack operations
- StackImplementation.java – contains stack-related logic
- BinarySearchTree.java – implementation of BST operations
- PostfixCalculator.java – evaluates postfix expressions
- Main.java – entry point for running the program


Execution
1. Compile the source files
2. Run the Main class
